
                            How to run code

Step 1: Navigate to project then click on database folder and download sql file to workbench
Step 2: Open project on NetBeans
Step 3: download both Gson, and mysql-connecter
Step 4: navigate to Services tab. 
step 5: navigate to Servers->Apache Tomcat->properties 
step 6: navigate to platforms tabs
Step 7: on VM options put: 

-DRDS_HOSTNAME=“yourhostName” -DRDS_PORT=“yourPort” -DRDS_FINDER_DB_NAME="Finder" -DRDS_USERNAME=“yourUserName” -DRDS_PASSWORD=“yourPass”
